# 词萌星球：505词离线网页版

## 语义配图版

单词卡已升级为原创“画面即释义”情境图：名词突出物体，动词展示动作，形容词使用对比，方位词显示位置关系。505个例句均为入门短句，并优先复用本词表旧词。

如果曾打开过旧版，请关闭原网页后重新双击启动文件；新版启动地址带有 `v=2`，会自动绕过旧缓存。

## 推荐打开方式

### macOS

双击 `启动离线版.command`，浏览器会自动打开。学习期间保持终端窗口开启，结束后按 `Control + C`。

### Windows

双击 `启动离线版.bat`。如果电脑安装了Python，会通过本机 `localhost` 打开；否则自动打开直接离线版。

### 直接打开

也可以直接双击 `index.html`。505个词及其发音均保存在本文件夹中，不需要联网。部分浏览器对本地文件的麦克风权限管理较严格，此时改用上面的一键启动方式。

## 离线录音

1. 点击单词卡下方的“跟读录音”。
2. 第一次使用时允许浏览器访问麦克风。
3. 录音约2.6秒后自动停止。
4. 页面显示本地播放器，可立即听自己的发音。
5. 录音仅保存在当前浏览器页面的内存中，不上传；关闭页面后录音音频自动释放，但“已完成跟读”状态保留在本机。

## 功能

- 完整505词、101课，每课5词。
- 505份离线发音。
- 本地录音和即时回放。
- 五种检测、课程解锁、错题重测。
- 学习达标后获得钥匙，未学习不能开盒。
- 10套共60个潮玩角色、重复角色转星尘。
- 全词中英文搜索、收藏和成长记录。
- 学习进度保存在浏览器 `localStorage`，不上传。

## 手机离线使用

将本文件夹部署到HTTPS网站后打开，选择“添加到主屏幕”。首次打开会缓存505个发音文件，缓存完成后可断网使用。直接把文件夹传到手机时，手机浏览器对本地网页文件的支持差异较大，推荐使用HTTPS安装型PWA。

## 注意

- 本版实现离线录音与回放，不包含离线音素级自动评分。
- 清理浏览器网站数据会同时清除学习进度。
- 485词基础内容由本机中英词典生成，正式教学发布前建议继续逐词教研复核。
